# Security Policy

I use [Socket](https://socket.dev/) and [Deepsource](https://deepsource.com/) for this project.

## Cryptography Dependencies

- golang.org/x/crypto for core cryptographic primitives
  - hkdf
  - curve25519

- go/crypto
  - ed25519
  - sha256
  - rand
  - aes
  - cipher
  - hmac
